# Build

Use this skill whenever source files in `src/`, `_partials/`, or `site.config.json` change and output should be regenerated in `dist/`.

## Inputs

- Project root with `src/`, `_partials/`, and `site.config.json`
- Tailwind output at `src/css/site.css` (typically generated by `npm run build:css`)

## Steps

1. From the project root, run `npm run build`.
2. If only the core build step is needed, run `skilled-build --project-root .`.
3. Confirm output files exist in `dist/`.

## Output

- `dist/` refreshed from `src/`
- Header/footer partials injected in built HTML files
- SEO/canonical/Open Graph metadata injected from `<!-- meta -->` blocks
- `dist/sitemap.xml` and `dist/robots.txt` regenerated
